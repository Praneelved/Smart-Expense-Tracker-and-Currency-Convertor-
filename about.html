ℹ️ About This Project

Smart Expense Tracker & Currency Converter — A Micro Project for college submission.

🎯 Project Objective

To develop a web-based application that helps users track their daily expenses by category, manage their monthly salary and remaining balance, and convert between different world currencies — all while promoting financial literacy and responsible consumption habits aligned with UN Sustainable Development Goals 8 and 12.

✨ Key Features
💼
Salary Input & Balance Tracking

Enter monthly salary and see remaining balance update automatically as expenses are added.

📂
Category-wise Expense Tracking

Add expenses under categories — Food, Shopping, Travelling, Medical, Entertainment, Bills, Education and Other.

📊
Category Summary with Progress Bars

Visual breakdown showing percentage of spending in each category with colored progress bars.

💱
Live Currency Converter

Convert between 150+ world currencies using live exchange rates fetched from ExchangeRate API.

💾
Data Persistence with localStorage

All expenses and salary data are saved in the browser's localStorage — data survives page refresh.

🌍
SDG Mapping

Project is mapped to UN SDG 8 (Economic Growth) and SDG 12 (Responsible Consumption).
